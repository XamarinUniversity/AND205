Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This course introduces three common user-interface navigation metaphors for Android: tab, gestural, and drawer. It also shows how to combine tab and gestural to give the user options on how they switch between views. These patterns typically use Fragment and FrameLayout so the course begins with a quick overview of how those work.

Objectives

  1. Use Fragment and FrameLayout to swap views
  2. Implement tab navigation with TabLayout
  3. Implement gestural navigation with ViewPager
  4. Combine gestural and tab navigation
  5. Implement drawer navigation with DrawerLayout

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Use Fragment and FrameLayout to swap views
    Exercise
    This lab lets you practice with two fundamental concepts used in almost all Android navigation patterns: Fragment and FrameLayout.
  2. Implement tab navigation with TabLayout
    Exercise
    The goal of this lab is to code the tab navigation pattern.
  3. Implement gestural navigation with ViewPager
    Exercise
    Here you will code gestural navigation to let the user swipe to switch between views.
  4. Combine gestural and tab navigation
    Exercise
    This exercise combines the previous two techniques to enable both tab and gestural navigation inside a single Activity.
  5. Implement drawer navigation with DrawerLayout
    Exercise
    The lab shows how to code Drawer navigation to let the user switch directly to any app view.
